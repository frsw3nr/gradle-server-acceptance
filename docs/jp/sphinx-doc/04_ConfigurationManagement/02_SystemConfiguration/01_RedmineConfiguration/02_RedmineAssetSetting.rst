Redmineトラッカー
-----------------

管理対象設備を類型化したものを、Redmineトラッカーとして定義します。
各管理対象はトラッカーごとにチケットを登録します。
以下のトラッカーを定義します。

設備トラッカー
^^^^^^^^^^^^^^

* IAサーバ

   Linux、Windows、ESXi サーバ用

* UNIXサーバ

   Solaris、AIX、HP-UX など UNIX サーバ用

* ストレージ

   共有ストレージなどストレージ機器

* ネットワーク

   ルータ、スイッチなどネットワーク機器

* ソフトウェア
* その他設備

プロジェクト管理用トラッカー
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

* 課題

   課題、WBS、作業用

* インシデント
