Changes for 0.1.2 (2016-11-7)
--------------------------------------------------------------------------------

New features:
* vCenter 用 PowerShell スクリプトに例外処理追加、初回のログイン処理に失敗したら終了する
* Windows 用 PowerShell スクリプトに Enter-PSSession コマンドを追加して、セッションを保持する様に変更
* 検査結果がnull(空欄)の場合はエビデンスの結果セルをNGに配色する

Bug fixes:
* PowerShell の例外処理発生時にエラーが出力されない問題対処
* Linux 検査のSSHセッション例外発生時にプログラムが終了する問題対処

Changes for 0.1.1 (2016-10-31)
--------------------------------------------------------------------------------

New features:
* 検査ID絞り込みオプションを追加
  getconfig -s {サーバ名} -t {検査ID} とすることで指定した検査のみ実行
* 検査対象シートを横持から縦持ちに変更。設定項目を追加して検査シナリオで設定値を使用出来る様に変更
* test.serialization.tasksで並列化させないタスクを指定。
  vCenter情報採取などのケースで多重化した場合に検査対象側サーバに負荷が集中する場合に指定する
* Linux,Windows,vCenterの不具合検査項目を修正
* ドキュメント 使用方法、開発者ガイドを作成

Bug fixes:
* SSH ワークディレクトリ削除エラーの例外対処
* 非仮想化サーバでvCenter検査が実行されてしまう問題対処

New features:
* 初版作成
